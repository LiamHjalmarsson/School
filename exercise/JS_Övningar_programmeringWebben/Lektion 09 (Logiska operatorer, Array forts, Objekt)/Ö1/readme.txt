Övning 1
========

I denna övningen ska ni utgå från den array (av användare) som finns i
"main.js". Ni ska skriva ut (console.log) de egenskaper som ni finner i listan
nedan. Detta är en övning i att läsa av en större mängd data - som ni kommer
upptäcka är objekten i "main.js" ganska stora (vilket är väldigt vanligt).

I exempeldatan kan ni också se ett alternativt sätt till hur vi kan skriva
egenskaper i ett objekt. Antingen med eller utan citationstecken, båda gör samma
sak i JavaScript. Dvs. dessa två exempel innebär samma sak:

    let user = { name: "Sebbe" };   // Ex1
    let user = { "name": "Sebbe" }; // Ex2

Ut i det vilda (google) kommer ni söta på båda varianterna, därav är denna
"nya" varianten med i övningen så att ni har stött på både och.

Skriva alla er utskrifter (console.log) i "main.js" efter all exempeldata.

Tips: glöm nu inte att utforka via konsollen i webbläsaren.
